## Problem Statement

Crime analysis is an important aspect of public safety and planning. By using historical crime data, machine learning models can identify trends and help in predicting future crime patterns. This project applies supervised machine learning techniques to analyze and predict crime rates.

## Dataset

The dataset contains historical crime-related data with features such as:

* Crime type
* Location
* Time-based attributes (year, month)
* Crime frequency

The dataset was cleaned and preprocessed before training the machine learning model.
